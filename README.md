注意事项：

一、电机有可能使电源电压不稳，导致蓝牙重启断联甚至单片机重启，建议重要模块与电机分开供电，再使用一个18650锂电池经过稳压3.3V给PCB板子上的模块供电，只要保证两个电源共地即可（即负极连接在一起）。

二、2.54mm排座可能不能很好的固定模块，比如HC-04蓝牙模块、电机编码器双头线仍可能松动，可以另想办法固定（比如直接焊接、用胶带固定等）。

三、小车身上的任何部件偏移导致整体重心偏移，都可能使最适PID参数变化，所以最好使所有部件固定牢固。
